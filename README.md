Котики захватят мир! (Ну или хотя бы избавят вас от проблем с покупкой домена). Тем кому нужны домены без лишних регистраций и подтверждений. Быстрое пополнение через криптовалюту и мгновенное использование собственного домена!

## Возможности бота:
1. Мгновенная покупка домена
2. Возможность пополнения баланса через криптовалюту (CrystalPay)
3. Передача домена другому пользователю
4. Автоматическая привязка к Cloudflare
5. Админ панель для пополнения баланса/созданию промокодов
6. Регулируемая маржа с покупки домена (начальная 40%)
7. Уведомления и логирование всех событий
8. Привязка NS сразу с покупкой домена.
